Tên dự án: Mini game giới thiệu khóa học Python

--------------------------------------------------------------------------------------------------------------------------------------------------

CÁC CHỨC NĂNG CHÍNH CỦA GAME
Game có trang menu và khi vào bên trong thì có trang lựa cho các level giúp các học viên biết mình đang ở level nào và có thể chơi tiếp tục màn ở lần
sau.
Có cơ chế nhặt trái cây và dùng nó đẻ mở khóa nhân vât mới

---------------GAMEPLAY---------------
Game được chia thành 2 loại xen kẽ.
- Loại 1: game không cho sử dụng bàn phím để di chuyển chỉ có thể di chuyển được nhờ code có thể nhập bằng IDE. Và bên dưới có các gợi ý code để qua màn.
Có hướng dẫn code đó được dùng như thế nào. Sau khi nhấn nút run thì nhân vật sẽ di chuyển theo ý mình.
- Loại 2: game sử dụng bàn phím và không cho sử dụng code. Giúp học viên có thể tự do hơn không bị gò bó trong việc code. Và giúp xác định khoảng
cách cũng như có thể lấy được cảm giác chơi game khiến cho dự đoán code và số lượng bước đi khi qua màn code dễ dàng hơn.
